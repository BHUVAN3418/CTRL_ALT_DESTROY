\# RainGuard AI 🌧️



\## Problem

Delivery workers lose income during rain.



\## Solution

Parametric insurance that auto-pays based on weather.



\## How it works

Worker registers → rain detected → payout sent



\## Premium

₹20–₹50/week



\## Triggers

\- Rainfall >35mm

\- IMD alerts

\- AQI >300

\- Heat >43°C

\- Platform outage



\## AI

Risk scoring + fraud detection



\## Tech

FastAPI, HTML, ML
